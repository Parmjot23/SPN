# SPN Logistics Website

This repository contains a lightweight static website for SPN Logistics located in Les CÃ¨dres, QC. It uses React via CDN with Tailwind CSS for styling, GSAP for simple animations, and EmailJS for sending form submissions.

Open `index.html` in your browser to view the site. Update the EmailJS `YOUR_PUBLIC_KEY`, `YOUR_SERVICE_ID`, and `YOUR_TEMPLATE_ID` values in `js/app.js` to enable contact forms.
